# 🔄 Daily Prompt Automation (Dataverse + Power Automate + Copilot)

This project demonstrates how to build a daily task automation system using:

- **Microsoft Dataverse**
- **Power Automate Cloud Flow**
- **Microsoft Copilot (Power Platform Copilot)**
- **Automated emails + Dataverse row updates**
- **Daily recurrence schedules**

The solution includes the Dataverse table, the Cloud flow, the prompt list used for generation, and screenshots of the workflow.
